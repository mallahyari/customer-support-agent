Setup Instructions

  1. Start the backend server:
    2. cd backend && uvicorn main:app --reload --port 8000
  2. Create a test bot: Use the admin API or database to create a bot and note its id and api_key
  3. Update the widget configuration below with your bot ID and API key
  4. Open this HTML file in a browser
  5. Click the chat button in the bottom-right corner to test!
üí° Note: The widget loads from dist/widget.js which was built using npm run build. If you make changes to the widget code, rebuild it before testing.

Widget Configuration

<script src="dist/widget.js"></script>
<script>
  ChirpWidget.init({
    botId: 'YOUR_BOT_ID',
    apiKey: 'YOUR_API_KEY',
    apiUrl: 'http://localhost:8000'
  })
</script>

Replace YOUR_BOT_ID and YOUR_API_KEY with your actual bot credentials
